RACK

WHAT IS IT

Rack is a static library which include classes for work with flat-structured packages.
The library include:
  • Class 'RFile' - control a file data in a package
  • Class 'RDirectory' - control a directory data in a package
  • Class 'RPackage' - control a package data
The library availalbe for Windows only, no plans to port it. Rack package files have a flat structure - no file attributes, only a relative path and file data.

PROJECT GOAL

The main goal - create a very simple packager for own goals.
